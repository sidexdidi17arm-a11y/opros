# 📊 Приложение для статистики опроса по ФЭС ПАО "Россети-Юг"

Веб-приложение для мониторинга и анализа показателей опроса приборов учёта с хранением данных в PostgreSQL.

## 🌟 Особенности

- 📈 Визуализация статистики опроса по ФЭС
- 💾 Хранение данных в PostgreSQL
- 📊 Графики и диаграммы
- 📄 Генерация отчетов
- 🔐 Ролевая модель доступа
- 📱 Адаптивный дизайн
- 🔄 Сравнение данных по неделям

## 🚀 Быстрый старт

### Предварительные требования

- Node.js 16+
- PostgreSQL 12+
- npm или yarn
